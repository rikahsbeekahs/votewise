# VoteWise AI | Election Intelligence Guide 🗳️🤖

**VoteWise AI** is a premium, AI-powered interactive guide designed to empower Indian voters with accurate information, debunk election myths, and simplify the democratic process.

## 🌟 Key Features
- **Hybrid AI Assistant**: Powered by **Google Gemini 1.5 Flash**, with a robust **Local Knowledge Base** (40+ topics) for instant offline support.
- **Multilingual Support**: Fully accessible in **13 major Indian languages** (Hindi, Bengali, Telugu, Tamil, etc.).
- **Election Quiz**: 25+ interactive questions to educate first-time voters on constitutional rights and voting procedures.
- **Myth Buster**: A dedicated module to combat misinformation with verified facts regarding EVMs, VVPAT, and registration.
- **Live Pulse Simulation**: A real-time data visualization dashboard showing simulated voter participation metrics.
- **Accessibility First**: Designed with a clean, high-contrast white-and-blue theme for maximum readability.

## 🛠️ Technology Stack
- **Core**: HTML5, Vanilla JavaScript (ES6+), CSS3 (Modern Flexbox/Grid).
- **AI/ML**: Google Gemini API integration for natural language understanding.
- **Icons**: Lucide Icons for a modern, crisp UI.
- **Animations**: Custom CSS keyframes for smooth staggered entries and interactive pulses.

## 🚀 Impact
VoteWise AI addresses the information gap in the Indian democratic system by providing a non-partisan, secure, and user-friendly platform for civic education. By debunking myths and simplifying registration steps, it encourages higher voter turnout among the youth and marginalized communities.
